# Sistema Especialista para Diagnóstico de Falhas Automotivas 🚗🛠️

Este repositório contém o projeto prático desenvolvido para a disciplina de **Laboratório de Programação (2026/1)**. O objetivo é implementar um **Sistema Especialista** capaz de identificar falhas em veículos com base em sintomas fornecidos pelo usuário, utilizando três paradigmas de programação diferentes.

## 📐 Estrutura do Projeto

O sistema foi mapeado uniformemente para tratar **10 sintomas** e inferir **7 diagnósticos específicos**. A árvore de decisão lógica foi distribuída nas seguintes pastas:

```text
LaboratorioDeProgramacao/
└── Sistema_Especialista/
    ├── python/   # Paradigma Orientado a Objetos (Menus interativos)
    ├── lisp/     # Paradigma Funcional (Manipulação de listas/símbolos)
    └── prolog/   # Paradigma Lógico (Motor de inferência e unificação)
